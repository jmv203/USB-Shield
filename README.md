 Shield-Port AI: Zero-Trust USB DefenseShield-Port AI es un ecosistema de defensa avanzado diseñado para neutralizar ataques de Juice Jacking. Actúa como un "firewall inteligente" situado entre un puerto de carga potencialmente comprometido y el dispositivo del usuario.🚀 El Problema: Juice JackingLos puertos USB en aeropuertos o cafeterías pueden ser manipulados para inyectar malware o extraer datos sin que el usuario lo note, aprovechando que los cables transmiten energía y datos simultáneamente. Aunque los móviles modernos tienen barreras, dispositivos como cámaras o e-readers son vulnerables , y la ingeniería social puede engañar incluso al usuario más precavido.
 🧠 La Solución: El "Triple Combo" de DefensaEste proyecto utiliza un Gemelo Digital para validar una arquitectura de seguridad basada en tres pilares:Criptografía Asimétrica (RSA): Implementación de un protocolo Challenge-Response. El sistema exige una firma digital dinámica que solo un hardware legítimo puede proporcionar, eliminando ataques de clonación.Inteligencia Artificial Conductual: Un modelo de Machine Learning (Isolation Forest) analiza la geometría del tráfico de datos en tiempo real.Análisis de Entropía: El sistema utiliza la Entropía de Shannon para detectar anomalías estructurales y ofuscación de malware:
$$H(X) = - \sum_{i=1}^{n} P(x_i) \log_2 P(x_i)$$🛠️ Laboratorio de SimulaciónEl proyecto se ha validado en un entorno controlado con tres nodos:Atacante: Simula puertos comprometidos y ataques de inyección.Escudo: El nodo central que ejecuta los algoritmos de detección.Protegido: El receptor que monitoriza la salud de la conexión en tiempo real.📖 ¿Quieres saber más?Si quieres profundizar en los detalles técnicos, ver las fórmulas matemáticas en acción o entender cómo funciona la simulación del Gemelo Digital, echa un vistazo a:👉 notes.ipynb 👈Ahí encontrarás la explicación detallada de cada fase del proyecto, las métricas de varianza y cómo la IA toma decisiones para proteger tus datos
